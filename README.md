Um script simples do Python que Utiliza BeautifulSoup e Requests + Funções built in do python para realizer Web Scraping de Livros no Site https://books.toscrape.com

Você escolhe uma nota de 1 a 5 e o script vai percorrer as primeiras 10 páginas do Site e buscar todos os livros com a nota selecionada, depois te recomendará um deles aleatóriamente.
